南开学子跻身QP案例分析全国赛三强

来源： 南开新闻网     发稿时间： 2017-12-12 17:29

　　南开新闻网讯(通讯员 芦聪 赵洪昌)近日，由香港会计师公会主办的“Qualification Program Case Analysis Competition”在上海举办。由南开大学商学院本科生卓德麟、孟曹伟、任珈萱以及经济学院本科生廖燕芳组成的南开大学代表队晋级全国决赛，最终摘得全国季军和最佳书面报告两项荣誉，再创我校学子在该项赛事的历史新高！

　　“Qualification Program Case Analysis Competition”是由香港会计师公会发起，旨在帮助在校生发展商业技能、培养职业判断、为面对商业环境做好准备的高校赛事。今年赛事吸引到清华大学、中央财经大学、南开大学等436支队伍参赛。决赛阶段，南开大学代表队得到了会计系姚颐教授指导，商院团委鲍文海担任带队老师。最终我校代表队凭借流畅的英文表达、扎实的财务分析、切实的实施方案、完美的报告展示获得在场评委的一致认可，最终跻身全国三强。

　　近年来，商学院团委拥抱“双创”，与学院专业系所通力合作，打造系列学科赛事平台，以赛促学，培养学生创新能力。全年主办或协办学科竞赛十余次，既有KPMG、QP案例分析大赛等全国高校商业应用赛事，又有SODA数据应用大赛等服务社会热点的社会性专业创新赛事，还有“尖烽时刻”商业模拟大赛等全球性专业赛事，斩获多项国内及国际竞赛大奖，为母校争得荣誉。
